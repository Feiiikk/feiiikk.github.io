# Transition Tracker

Open `index.html` directly for local-only use. To enable private cross-device sync, follow `DEPLOY.md`.

The application remains dependency-free in local mode. Cloud mode loads the official Supabase browser client only after Supabase is configured.

## Version 2 experience

- Create a course and category without leaving a study entry.
- Type an existing course or a brand-new course in the same searchable field; new courses are created when the session is saved.
- The course picker now uses a consistent searchable dropdown across Safari, Chrome, desktop, and mobile; emoji are display-only and never need to be typed.
- Choose and revise category names and emoji instead of accepting an automatic guess.
- Use modules, lessons, chapters, weeks, sessions, or open-ended progress.
- Start from friendly onboarding and quick logging instead of a dense setup flow.
- Keep secondary metadata under **More details**.
- Install the hosted tracker on a phone or tablet as a home-screen web app.
